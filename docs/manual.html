Scouting Report 使い方マニュアル

チーム共有のパフォーマー名簿アプリ ─ ブラウザだけで使えます（アプリのインストール不要）

📍 https://aratama-ship-it.github.io/scouting-report-app/

このアプリでできること

  • 登録済みパフォーマー（約225組）の名簿を検索・閲覧
  • 各アーティストの詳細ページで、似たスキルを持つ関連アーティストを発見
  • 気になるアーティストにお気に入り ☆ を付けてチームで共有
  • アーティストごとにコメント（メモ）を残してチームで相談
  • 毎週自動調査される新規候補や、名簿の変更履歴・集計を確認

はじめかた（初回アクセス）

1上のURLをスマホまたはPCのブラウザで開く

2チームの合言葉を入力して「開く」（合言葉は管理者から直接共有されます。このマニュアルには書きません）

3お名前（ニックネーム）を入力。お気に入りやコメントに表示される名前です

💡 2回目以降: 合言葉と名前はブラウザが覚えるので、URLを開くだけですぐ名簿が表示されます。ブックマーク／ホーム画面に追加しておくと便利です。

画面の見かた ─ 4つのタブ

名簿登録アーティストの一覧。検索ボックス（名前・スキル・メモ）とカテゴリ絞り込みが使えます。カードをタップすると詳細ページへ。
候補毎週の自動調査で見つかった新規スカウト候補。「シート追記済み / 未追記」のタグで名簿への反映状況がわかります。
履歴名簿の週ごとの変化（追加・削除・変更）の記録。
集計登録数とカテゴリ別の内訳グラフ、スカウトが手薄なジャンルのメモ。

詳細ページでできること

お気に入り ☆

「☆ 0人」ボタンを押すと自分のお気に入りとして登録され、★に変わります。数字はチーム全体で何人が☆したか。もう一度押すと取り消せます。☆が付いたアーティストは名簿一覧でも★マークが表示されるので、チームの注目株がひと目でわかります。

コメント

「コメントを入力」欄に書いて「投稿」を押すだけ。あなたの名前と日時つきでチーム全員に表示されます。「この人、来年の公演に合いそう」「連絡済み・返信待ち」などの共有メモにどうぞ。他の人の書き込みを見るときは「更新」ボタンで最新に。

⚠️ 注意: コメントは今のところ削除ボタンがありません（管理者がスプレッドシート側で削除できます）。誤字は気にせず、書き直しでOKです。

名前（ニックネーム）の変更

画面右上の「👤 名前」ボタンを押すといつでも変更できます。

よくある質問

スマホでも使えますか？
使えます。スマホ・タブレット・PCどれでもOK。画面は自動で最適化されます。
合言葉を忘れました
管理者（Arata）に直接聞いてください。メールやSNSに書き残さないようお願いします。
「ロック」ボタンは何ですか？
ログアウトのようなものです。共用PCで見たあとなど、他人に見られたくないときに押してください。次回また合言葉が必要になります。
InstagramやYouTubeのリンクを押すとどうなりますか？
新しいタブでそのアーティストのページが開きます。アプリの画面はそのまま残ります。
名簿のデータはいつ更新されますか？
週1回（月曜朝）の自動更新です。画面上部に「◯◯◯◯-◯◯-◯◯時点」とデータの基準日が表示されています。
お気に入りやコメントが表示されません
通信の一時的な問題の可能性があります。「更新」ボタンか、ページの再読み込みを試してください。名簿の閲覧自体はそのまま使えます。

管理者向けメモ（Arata用）

名簿データ更新週次定期タスクが自動実行（スナップショット保存 → npm run build → push → Pages反映）。手動更新も同じ手順。
☆・コメントの実体スプレッドシートの Favorites タブ。不適切な行は直接削除してOK（即反映）。
候補の自動追記node tools/push-candidates.mjs <候補md> → シートの Candidates タブへ。同名は自動スキップ。採用する候補は本名簿の行へ手で移す。
合言葉の変更① .secret を書き換え → npm run build → push　② GASのスクリプトプロパティ PASSPHRASE も同じ値に変更　③ チームに新しい合言葉を連絡。
GASコードの更新gas/DEPLOY.md の手順6（エディタに貼り直し → デプロイを管理 → 新バージョン。URLは変わらない）。
コミット禁止物生の名簿md・シートID・合言葉（.secret）は公開リポジトリに入れない。
